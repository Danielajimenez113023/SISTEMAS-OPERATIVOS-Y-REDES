Lista desordenada

Ordene cada ítem, según su comprensión; mueva cada ítem según su orden lógico.

  • El término carpeta se utiliza como sinónimo de directorio.
  • La operación Create se utiliza para crear un nuevo directorio o carpeta en el sistema.
  • Para eliminar un archivo o carpeta se utiliza el comando Delete.
  • La operación Opendir permite abrir un directorio y leer su contenido en Linux.
  • Para cerrar un directorio abierto en Linux se utiliza Closedir.
  • La operación Readdir permite listar el contenido de un directorio.
  • Si se desea cambiar el nombre de un archivo o carpeta se usa Rename.
  • La operación Link permite crear un enlace (link) a un archivo o carpeta.
  • Para eliminar un enlace a un archivo se utiliza Unlink.
